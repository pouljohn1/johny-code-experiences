JSX lets you write HTML-like syntax inside JavaScript, and props let you customize components with data and behavior.

## What Is JSX
JSX is syntax sugar for `React.createElement` calls.

```jsx
const title = 'React';
const heading = <h1>Hello, {title}!</h1>; // JSX expression
```

- Use `{}` to embed JavaScript expressions
- Use `className` instead of `class`
- Inline `style` takes an object: `style={{ color: 'tomato' }}`

## Embedding Expressions
You can compute values inside JSX.

```jsx
const user = { first: 'Ada', last: 'Lovelace' };

function FullName({ user }) {
  const name = `${user.first} ${user.last}`;
  return <strong>{name.toUpperCase()}</strong>;
}
```

## Passing Props
Props are attributes you set on a component.

```jsx
function Badge({ label, color = 'gray' }) {
  // default value for color
  return <span className={`badge badge-${color}`}>{label}</span>;
}

export default function App() {
  return (
    <div>
      <Badge label="New" color="green" />
      <Badge label="Beta" /> {/* color defaults to gray */}
    </div>
  );
}
```

- Destructure props in the function parameters and provide defaults
- Strings can be written as `prop="value"`; expressions go in `{}`

## children Prop
`children` lets a component receive nested content.

```jsx
function Panel({ title, children }) {
  return (
    <section className="panel">
      <h3>{title}</h3>
      <div>{children}</div>
    </section>
  );
}

export default function App() {
  return (
    <Panel title="About">
      <p>Panels can wrap any content via children.</p>
    </Panel>
  );
}
```

## Passing Functions as Props
Lift behavior into parent components and pass callbacks down.

```jsx
function Button({ onClick, children }) {
  return <button onClick={onClick}>{children}</button>;
}

export default function App() {
  const handleSave = () => {
    console.log('Saved!');
  };
  return <Button onClick={handleSave}>Save</Button>;
}
```

## Styling in JSX
- `className` for CSS classes
- Inline styles use camelCased properties

```jsx
function Tag({ text }) {
  return (
    <span className="tag" style={{ backgroundColor: '#eef', padding: 4 }}>
      {text}
    </span>
  );
}
```

## Spreading and Rest Props (Use Carefully)
Sometimes you want to pass unknown props through.

```jsx
function Input({ label, ...inputProps }) {
  return (
    <label>
      <span>{label}</span>
      <input {...inputProps} />
    </label>
  );
}
```

- Prefer explicit props for clarity; spread only when it improves ergonomics
