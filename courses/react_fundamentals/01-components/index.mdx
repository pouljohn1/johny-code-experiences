## Tips for Success

* Keep components small and focused
* Prefer composition over deep prop drilling
* Name components in PascalCase and files in a predictable structure

## Common Pitfalls

* Returning multiple elements without a wrapper (use a fragment `<>...</>`)
* Using `class` instead of `className` in JSX
* Forgetting to add a `key` when rendering lists
