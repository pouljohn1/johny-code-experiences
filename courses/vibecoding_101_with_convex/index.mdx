Welcome to **VibeCoding 101**! In this course you’ll learn how to:

* Describe an app idea in plain English
* Let **Convex Chef** turn it into a working web app
* Refine it with simple chat prompts
* Ship and share it with others

You don’t need any programming background—if you can explain what you want, you can build.
