### Step 1: Getting Started with v0

1. Open [v0.app](https://v0.app) in your browser
2. If you don't have an account, click the **Sign Up** button to register
3. Complete the registration process

### Step 2: Creating Your First Web App

Once you're signed in, you'll see the v0 main page with an input field in the center. This is where you'll write prompts to create your web app.

**Example prompt to try:**
```
Build me a landing page for my startup. My startup uses AI to help people learn new things. The design needs to be sleek with lots of modern effects.
```

After sending your prompt:
- The AI will begin writing code in real time
- Within moments, you'll see your first AI-generated landing page
- The page is fully functional and ready to customize

### Step 3: Understanding the v0 Interface

**Left Sidebar Tabs:**
- **Chat** - Interact with the AI to make changes and improvements
- **Design** - Customize fonts, colors, and visual elements
- **Connect** - Hook up external data sources (covered in later modules)
- **Vars** - Set custom variables for data sources (covered in later modules)
- **Rules** - Write custom instructions for the AI agent
- **Settings** - General configuration options

**Center Panel:**
- **Preview Mode** - See your website as users will see it
- **Code Mode** - View the complete source code (toggle using the toolbar icons)
- The preview is fully interactiveâ€”try clicking navigation links to see it behave like a real website

### Step 4: Publishing Your Web App

1. Click the **Publish** button at the top of the screen
2. In the **Customize Domain** field, enter your preferred subdomain name (e.g., `learn-ai-with-experts`)
3. Leave other options at their default settings for now
4. Click **Publish to Production**
5. Your website is now live on the internet!
