Nuxt.js is a framework on top of Vue that makes building modern web apps easier, especially for SSR, routing, and DX.

## What you will learn

- How to create a new Nuxt app
- Basic project structure
- How to run it in development and build for production

## Prerequisites

- Node.js 18+ installed
- Basic familiarity with JavaScript and Vue (helpful but not mandatory)

## 1. Create a new Nuxt project

From a terminal, run:

```bash
npx nuxi@latest init my-nuxt-app
cd my-nuxt-app
npm install   # or pnpm install / yarn
```

This scaffolds a minimal Nuxt project in `my-nuxt-app`.

## 2. Run the development server

```bash
npm run dev
```

Then open the printed URL, usually:

```text
http://localhost:3000
```

You should see the default Nuxt welcome page.

Create `pages/about.vue`:
